Windows Subsystem for Linux(WSL) allows a user to run Linux within a Windows environment.  Windows 10 and Windows 11 already have this capability built in.  This allows you to download and run several Linux Distros without affecting the Windows host operating system. 

A. Prerequisites
Windows 10 (1903+) or Windows 11 with WSL 2 support.
Administrative access to enable Windows features.
A working internet connection.
